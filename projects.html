Aleph-Nought Learning Platform ↗

A full-stack platform I built, with content covering six Dartmouth math and physics course syllabi. Its core is a symbolic exercise engine that parses and evaluates free-form mathematical expressions for automated grading, fed by a content pipeline I built (Manim-animated lectures, transcribed TA audio). Full-stack: Next.js, FastAPI, PostgreSQL, deployed to production.

QuBlitz Arena

Part of QuBlitz, a browser-based quantum lab built within the Fitzpatrick Lab. I built the QuBlitz Arena game — its mechanics and play — and the physics engine beneath it: I implemented and unit-tested the underlying quantum mechanics (phase interference, Bell-pair entanglement) against a passing automated test suite, so the simulation stays numerically honest.

Studium

A local-first AI study tool: uploaded PDFs become quizzes, flashcards, and concept maps, with a quantitative drilling module for timed practice. I kept it private by design — API keys in the OS keychain, user data on-device — and hardened the server myself, writing the SSRF and request-body-cap regression tests that gate the network layer.
